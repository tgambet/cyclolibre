Informations générales

CycloLibre est une application libre développée par Thomas Gambet.

L'application permet d'afficher une carte ou un tableau des stations de vélos, de visualiser rapidement l'encombrement ou la disponibilité d'une station en temps réel, d'effectuer une recherche, et de trouver une piste cyclable ou un itinéraire conseillé à vélo dans votre ville.

Pour tout contact envoyez-moi un email.

Informations techniques

Cette application est une application Web monopage (ou SPA) développée en utilisant le framework JavaScript Angular.

Le code source est hebergé chez Github et peux y être gratuitement consulté ou téléchargé.

Si vous avez un compte Github vous pouvez ouvrir un ticket pour rapporter un bug ou proposer une amélioration, "forker" le code, ou encore soumettre une pull-request.

La license d'utilisation de l'application est la license MIT.

L'instance principale de CycloLibre (cyclolibre.fr) est hébergée chez Heroku.

Les données de disponibilité des stations de vélos en libre service sont fournies par JcDecaux.

Les données géographiques et la position des pistes cyclables et itinéraires conseillés sont fournies par Google.
